什么是 Zelpis？ ​

Zelpis 是一个创新的前端渲染框架，通过 DSL (领域特定语言) 驱动的方式简化前端开发流程。它的设计理念是提供一个统一的接口，支持多种前端框架和渲染模式，让开发者可以更专注于业务逻辑而不是繁琐的配置和重复代码。

核心优势 ​

  • 灵活性：支持多种前端框架和渲染模式
  • 效率：通过 DSL 配置减少重复代码
  • 可扩展性：模块化设计，易于扩展和定制
  • 性能：基于 Vite，提供快速的开发体验
  • 类型安全：使用 TypeScript 开发，提供良好的类型支持

适用场景 ​

  • 需要同时支持 React 和 Vue 的项目
  • 需要服务端渲染的应用
  • 希望通过配置来驱动前端渲染的场景
  • 大型前端项目的模块化开发
